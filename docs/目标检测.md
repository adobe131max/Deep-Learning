# 目标检测 - Object Detection

## 算法

### RCNN

算法流程：

1. 由图像使用Select Search方法生成多个候选区域
2. 使用深度神经网络对每个候选区域提取特征
3. 根据特征使用SVM分类
4. 使用回归器修正候选框位置

### Fast-RCNN

算法流程：

1. 由图像使用Select Search方法生成多个候选区域
2. 使用深度神经网络对整个图像生成特征图，并将候选区域投影到特征图上获得特征矩阵
3. 对每个特征矩阵通过ROI pooling缩放，再通过全连接得到预测结果

### Faster-RCNN

算法流程：

1. 图像输入网络得到特征图
2. 使用 RPN 生成候选框，将候选框投影到特征图上获得特征矩阵
3. 对每个特征矩阵通过ROI pooling缩放，再通过全连接得到预测结果
